## Why this portfolio exists

Technology evolves quickly, and I believe the best way to understand new platforms is by building with them.

This GitHub documents that continuous learning process. Some repositories are proof-of-concepts, others are small utilities or technical experiments, while some evolve into more complete projects. Together, they reflect my approach to learning: start with fundamentals, experiment, measure, iterate, and document the results.

The goal is not simply to make things work, but to understand *why* they work, how they perform, and where they can be improved.
